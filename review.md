# Обзор проекта SmartPC (UX/UI)

---

# UX (опыт пользователя)

С точки зрения пользователя сайт воспринимается как помощник в выборе компьютера.

Основные пользователи:

- геймеры
- специалисты
- владельцы бизнеса
- люди, которым нужен ремонт


Для каждой группы есть свой путь:

- выбрать готовую сборку
- собрать ПК самостоятельно через конфигуратор
- заказать обслуживание


Главный плюс такого подхода — снижение сложности выбора.

Человеку не обязательно разбираться во всех комплектующих, так как сайт объясняет ценность готового решения.


---

# UI (визуальная часть)

Визуальный стиль SmartPC должен передавать:

- технологичность
- мощность
- современность
- надёжность


Ожидаемый стиль:

- современный тёмный интерфейс
- акцент на изображения сборок
- понятные карточки товаров
- визуальное отображение производительности


Дизайн должен напоминать игровую и IT-среду, но оставаться понятным для обычного пользователя.


---

# Впечатление нового пользователя

При первом посещении пользователь должен видеть:

- какие компьютеры продаются
- какие услуги доступны
- почему можно доверять компании


Важную роль играют:

- реальные фотографии сборок
- отзывы
- тесты
- гарантия


Это помогает убрать страх покупки дорогой техники.

# Сильные стороны проекта

## Понятная специализация

SmartPC не пытается продавать всё подряд.

Фокус:

- компьютеры
- комплектующие
- обслуживание


---

## Конфигуратор

Конфигуратор является одним из главных элементов проекта.

Он повышает вовлечённость пользователя и помогает ему самому участвовать в создании будущего ПК.


---

## Сервис после покупки

Проект не заканчивается продажей.

Пользователь может вернуться для:

- ремонта
- чистки
- обновления компонентов
