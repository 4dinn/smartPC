2. Техническое задание

Название проекта: интернет-магазин по продаже готовых ПК, комплектующих и услуг сервиса.
Тип проекта: коммерческий веб-сайт (интернет-магазин + сервисный центр).
Цель: продажа товаров и привлечение заявок на обслуживание.

2.2. Функциональные требования
1. Каталог готовых ПК

Отображение карточек сборок

Фильтрация по:
цене
видеокарте
процессору
назначению (игры / работа / офис)
Возможность сравнения сборок
Кнопка "Купить" и "В 1 клик"

2. Каталог комплектующих

Разделы:
видеокарты
процессоры
материнские платы
оперативная память
SSD / HDD
блоки питания
корпуса

Фильтры по бренду, цене, характеристикам
Сортировка по цене и популярности

3. Конфигуратор ПК

Пользователь выбирает комплектующие
Автоматическая проверка совместимости
Подсчёт итоговой стоимости
Кнопка оформления заказа

4. Раздел «Сервис и ремонт»

Список услуг:

диагностика
чистка
замена термопасты
апгрейд
Онлайн-заявка на ремонт
Форма обратной связи

5. Личный кабинет пользователя

Регистрация и авторизация
История заказов
Статус ремонта

6. Корзина

Добавление/удаление товаров
Изменение количества
Подсчёт итоговой суммы
Выбор способа доставки
Выбор способа оплаты

7. Админ-панель

Добавление/редактирование товаров
Управление заказами
Управление пользователями
Просмотр статистики продаж

2.3. Нефункциональные требования

Адаптивный дизайн (ПК / планшет / смартфон)
Защита данных пользователей
Интуитивно понятный интерфейс

2.4. Технологический стек

Frontend:

HTML
CSS
JavaScript (React / Vue)

Backend:

Node.js / PHP
База данных: MySQL или PostgreSQL

Хостинг:

VPS или облачный сервер
