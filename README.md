# SmartPC

**SmartPC** — учебный проект интернет-магазина по продаже готовых ПК, комплектующих и сервисному обслуживанию компьютеров.

## Возможности

* Главная страница
* Каталог готовых ПК
* Конфигуратор компьютера
* Каталог комплектующих
* Сервис и ремонт
* Блог
* Контакты и форма обратной связи

## Технологии

* React
* Vite
* JavaScript
* CSS

## Структура проекта

```text
app/      — исходный код React-приложения
design/   — макеты и дизайн проекта (PDF)
```

## Запуск проекта

```bash
cd app
npm install
npm run dev
```

Проект разработан в учебных целях.
